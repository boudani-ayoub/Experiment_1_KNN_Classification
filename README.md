# Experiment 1 — KNN Classification on Iris Dataset

## Overview
This experiment implements the **K-Nearest Neighbors (KNN)** algorithm to classify flowers in the Iris dataset using scikit-learn.

## What's Covered
- Loading and exploring the Iris dataset
- Exploratory Data Analysis (EDA) with scatter plots and correlation heatmaps
- Train/test splitting
- Training a KNN classifier
- Evaluating model accuracy

## Tech Stack
- Python 3
- scikit-learn
- matplotlib
- seaborn
- numpy

## How to Run
1. Install dependencies:
   ```bash
   pip install scikit-learn matplotlib seaborn numpy
   ```
2. Open and run the notebook:
   ```bash
   jupyter notebook KNN_Iris_Classification.ipynb
   ```

## Dataset
The Iris dataset is loaded directly via `sklearn.datasets.load_iris()` — no external download needed.
